# Автоматизация тестирования QIWI API

В данном проекте продемонстрирован подход к проектированию тестов на основе документации QIWI Wallet API.

## Архитектура:
   - `config.py` — управление конфигурацией и переменными окружения (.env).
   - `api_client.py` — изолированный API-клиент.
   - `test_main.py` — главный файл, содержащий в себе все тест-кейсы.

## Стек технологий
- **Язык:** Python 3.9+
- **Тестовый движок:** Pytest
- **HTTP-клиент:** Playwright (APIRequestContext)
- **Менеджер окружения:** Python-dotenv
